## UPDATE



#### SYNTAX:



UPDATE TABLE\_NAME

SET COL1=V1,COL2=VAL2,....COLN=VN

\[WHERE CONDITION] \[OPTIONAL]





//NON VEG --> 1-->PNAME--> BIRIYANI



UPDATE PRODUCT

SET PNAME="BIRIYANI"

WHERE PNAME="SUNSCREEN"; //IF NOT USE WHERE CONDITION IT WILL AFFECT TO ALL VALUES;



###### //TO UPDATE EXISTING DATA TO NEW DATA

UPDATE PRODUCT

SET PNAME="BIRIYANI"

WHERE PID=1;



###### //TO UPDATE TWO CLOUMNS DATA

UPDATE PRODUCT

SET PNAME="KABAB",PRICE=60

WHERE PID=5;







## DELETE 



IT IS USED TO DELETE EXISTING RECORDS FROM A TABLE



#### SYNTAX:



&#x09;DELETE 

&#x09;FROM TABLE\_NAME

&#x09;\[WHERE CONDITION];

## TRUNCATE



IT IS USED TO DELETE ALL THE RECORDS FROM A TABLE WITHOUT AFFECTING TABLE STRUCTURE.



###### SYNTAX:

&#x09;TRUNCATE TABLE TABLE\_NAME;



* TRUNCATE TABLE PRODUCT;









## DROP:

&#x20;IT IS USED TO DROP DATABASE AND ITS OBJECTS SUCH AS TABLE,VIEW,PROCEDURE,TRIGGER AND USER FROM MYSQL.





#### TO DROP TABLE FROM DATABASE:



SYNTAX:

&#x20;DROP TABLE TABLE\_NAME;





#### TO DROP DATABASE FROM MYSQL



SYNTAX:

&#x20;DROP DATABASE DATABASE\_NAME;
