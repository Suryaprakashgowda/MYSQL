#### Subquery:

\----------------



IT IS A QUERY INSIDE ANOTHER QUERY



CASE1: WHENEVER WE HAVE UNKNOWN VALUE IN THE QUESTION.





##### WORKING PRINCIPLE OF SUBQUERY:

\---------------------------------------------



> ALWAYS INNER QUERY EXCEUTES FIRST AND RETURN SOME OUTPUT.

>OUTPUT OF INNER QUERY WILL BE GIVEN AS INPUT FOR THE OUTER QUERY,

>OUTER EXECUTES COMPLETELY AND RETURNS FINAL OUTPUT.

> OUTER QUERY DEPENDS ON INNER QUERY.



##### CASE 1:

\-----------

WHENEVER WE HAVE UNKNOWN VALUE IN THE QUESTION WE USE SUBQUERY.



INNER QUERY:- TO FIND UNKOWN VALUES

OUTER QUERY:- TO PRINT FINAL OUTPUT.



##### RULES:-

##### \-------



1. THE CLOUMN NAME SELECTED IN THE INNER QUERY AND THE CLOUMN\_NAME WRITTEN INSIDE OUTER QUERY SHOULD BE OF SAME DATATYPE.



EX:- WHERE SAL>(SELECT FNAME   INVALID

&#x09;(INT)          (CHAR)



WHERE SAL>(SELECT COMM        VALID

&#x09;(INT)    (INT)

2\. WE CAN SELECT ONLY ONE CLOUMN INSIDE INNER QUERY.



EX:- WHERE SAL>(SELECT SAL,COMM     INVALID

##### TYPES OF SUBQUERY:

\---------------------------



1. SINGLE ROW SUBQUERY
2. MULTI ROW SUBQUERY
3. CO-RELATED SUBQUERY





##### 1.SINGLE ROW SUBQUERY

\--------------------------------



IF INNER QUERY RETURNS SINGLE VALUE THEN WE CAN CONSIDER QUERY AS SINGLE ROW SUBQUERY.



> IT SUPPORTS BOTH NORMAL OPERATIONS(=,!=,>,<) AND SPECIAL OPERATORS(IN,NOT IN,ALL,ANY)



##### 2\. MULTI ROW SUBQUERY

\----------------------------------

IF INNER QUERY RETURNS MORE THAN ONE VALUE THEN WE CAN CONSIDER WUERY AS MULTI SOW SUBQUERY.



> IT SUPPORT ONLY SPECIAL OPERATORS(IN,NOT IN,ALL, ANY)

##### ALL AND ANY:

\------------------

BOTH ALL AND ANY ARE MULTI VALUE OPERATORS WHICH TAKES MULTIPLE VALUES AT THE RHS AND SINGLE VALUE AT THE LHS ALONG WITH RELATIONAL OPERATIONS.

##### 

##### SYNATX:- ALL



COLUMN\_NAME/EXPRESSION < / > / <= / >= ALL (INNER QUERY OR V1,V2,V3, ---- VN)



##### SYNATX :- ANY



CLOUMN\_NAME/EXPRESSION  < / > / <= / >= ANY (V1,V2,V3,.... VN);





##### ALL

##### \---

##### IT IS A MULTVALUE OPERATOR WHICH TAKES MULTIPLE VALUES AT THE RHS AND SINGLE VALUE AT THE LHS ALONG WITH RELATIONAL OPERATORS(</>/<=/>=)

##### 

##### SYNTAX:

##### \--------



&#x20;          LHS                           RHS

COLUMN\_NAME/EXPRESSION </>/<=/>= ALL(V1,V2,....VN)



>IT WORKS ON AND CONDITION.



A:1000  T

B:2000  T

C:10000 F



SAL<ANY(2000,5000,10000)



1000<ANY(

2000<ANY(

10000<ANY(



##### ANY:

##### \----

##### IT IS A MULTIVALUE OPERATOR WHICH TAKES MULTIPLE VALUES AT THE RHS AND SINGLE VALUE AT THE LHS ALONG WITH RELATIONAL OPERATORS(</>/<=/>=)

##### 

##### SYNTAX:

##### \-------



COLUMN\_NAME/EXPRESSION </>/<=/>= ANY(V1,V2,.....VN)



>IT WORKS ON OR CONDITION.
