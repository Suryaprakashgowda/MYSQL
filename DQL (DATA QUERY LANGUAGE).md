##### Q. WQTD FANME,COMM WITH 15% HIKE AND ANNUAL SALARY WITH 50% DEDUCTION?



SELECT FNAME,COMM+COMM\*.15,(SAL\*12)-(SAL\*12)\*.50 FROM EMPS;



OR



SELECT FNAME,COMM+(COMM \* 15/100),(SAL\*12)-((SAL\*12)\*50/100) FROM EMPS;







##### Q.WQTD FNAME,JOB ANNUAL SALARY WITH 10 % HIKE AND ANNUAL COMMISSION WITH 20% DEDUATION??



SELECT FNAME,JOB,(SAL\*12)+(SAL\*12)\*.10,(COMM\*12)-(COMM\*12)\*.20 FROM EMPS;



OR



SELECT FNAME,JOB,(SAL\*12)+((SAL\*12)\*10/100) AS SALARY\_10\_HIKE,(COMM\*12)-((COMM\*12)\* 20/100) AS COMM\_10\_DEDUCTION FROM EMPS;



## **ALIASING:**

\-----------------------

IT IS PROCESS TO PROVIDE ALTERNATIVE NAME FOR A CLOUMN IN RESULTANT TABLE.





##### **RULES:**

\-------



###### > WITH OR WITHOUT USING KEYWORD WE CAN WRITE ALIAS NAME.



EXAMPLE:



SELECT SAL AS SALARY

FROM EMPS;



SELECT SAL SALARY

FROM EMPS;





###### > WE CAN USE MULTIPLE WORDS AS ALIAS NAME BY USING QUOTES OR BY CONNECTING THE WORDS WITH UNDERSCORE



EXAMPLE:



* &#x20;SELECT SAL\*12 AS ANNUAL\_SALARY FROM EMPS;



* &#x20;SELECT SAL\*12 "ANNUAL SALARY" FROM EMPS;



* SELECT FNAME,JOB,(SAL\*12)+((SAL\*12)\*10/100) AS "ANNUAL SALARY 10% HIKE",(COMM\*12)-((COMM\*12)\* 20/100) "ANNUAL COMMISSION WITH 10 % DEDUCTION" FROM EMPS;

## **DISTINCT**:



IT IS USED TO AVOID DUPLICATE VALUES FROM RESULTANT TABLE.



##### RULES:

\---------

* EITHER \* OR DISTINCT KEYWORD MUST BE THE FIRST ARGUMENT IN SLECT CLAUSE.



* WE CAN USE MULTIPLE CLOUMNS ALONG WITH DESTINCT KEYWORD BUT IT WILL AVOID COMBINATION OF DUPLICATE VALUES.



##### Q WQTD UNIQUE COMBINATION OF MGR AND STATUS?



SELECT DISTINCT MGR,STATUS

FROM EMPS;
